This project analyzes customer churn data to identify patterns and factors that influence 
customer retention. The goal is to help businesses understand why customers leave and improve 
retention strategies.


**The dataset contains customer information such as:**

1)Demographics
2)Account details
3)Services subscribed
4)Contract type
5)Churn status
(Source: Public telecom churn dataset)

 **What I Did**

1)Data cleaning and preprocessing
2)Handling missing values
3)Exploratory Data Analysis (EDA)
4)Data visualization using matplotlib & seaborn
5)Identified key churn drivers
6)Built insights for business decision-making

**Tech Stack**

1)Python
2)Pandas
3)NumPy
4)Matplotlib
5)Seaborn
6)Jupyter Notebook
